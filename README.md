# Dotfiles collection

## Overview

This contains various dotfiles intended for use in a linux environment. 

## Usage

To use these dotfiles, first clone this repository with your prefered method of cloning (I use ssh with `git clone git@github.com:Garry-Francis/dotfiles.git`). Then enter the directory of any of the files and read their respective `README.md` file for further instructions.
